# EV Charging Demand & Grid Stability Analysis

## Objective
Analyze EV charging demand patterns to identify peak usage hours
and provide insights for grid stability and energy optimization.

## Tools Used
- Power BI
- SQL
- Excel
- Python (Pandas, Matplotlib)

## Key Analysis
- Cleaned and prepared EV charging datasets
- Performed exploratory data analysis (EDA)
- Conducted time-series analysis to forecast demand
- Built interactive Power BI dashboards

## Key Insights
- Identified peak charging hours impacting grid load
- Highlighted opportunities for load balancing
- Suggested optimization strategies for renewable energy usage
